Подвиг 3. В программе объявлены два класса следующим образом:

class Money:
    __slots__ = '_money',

    def __init__(self, value):
        self._money = value


class MoneyR(Money):
    __slots__ = '_value',

И, затем, выполняются команды:

m = MoneyR(10)
m._money = 100
m._value = 20

Выберите все верные утверждения, связанные с этой программой.
