Подвиг 2. В программе объявлены два класса следующим образом:

class Money:
    __slots__ = '_money',

    def __init__(self, value):
        self._money = value


class MoneyR(Money):
    pass

И, затем, создается объект класса MoneyR:

m = MoneyR(10)
m.s = 100
